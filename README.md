# Java Engineering Fundamentals ☕

Repositório dedicado à construção de uma base sólida na linguagem Java e em Engenharia de Software.

O objetivo principal desta trilha é dominar o "Core Java", a Orientação a Objetos pura e as estruturas de dados fundamentais. Essa fundação é o pré-requisito para, em um segundo momento, avançar com total segurança e proficiência para o uso de frameworks corporativos (como Spring Boot) e o desenvolvimento de arquiteturas escaláveis.

## 🎯 Foco dos Estudos
- **Fundamentos da Linguagem:** Tipagem, coleções, manipulação de dados e segurança estrita do Java.
- **Orientação a Objetos Avançada:** Polimorfismo, interfaces, classes abstratas e encapsulamento real.
- **Engenharia de Software:** Clean Code, princípios SOLID e Design Patterns aplicados ao ecossistema Java.
- **Algoritmos e Estruturas de Dados (DSA):** Resolução de problemas com análise de performance e recursos.

## 🏗 Estrutura do Repositório
O aprendizado é organizado em módulos (`steps`), contendo implementações práticas, resolução de problemas e documentação técnica que compara as abordagens do Java com outros paradigmas.

- `step01_syntax_and_types`: Sintaxe base, operadores, controle de fluxo e sistema de tipos.
- *(Novos módulos serão adicionados conforme a progressão)*

## 🚀 Como Executar
Os códigos deste repositório utilizam Java puro e podem ser executados diretamente via CLI ou através de IDEs padronizadas da indústria (como IntelliJ IDEA).

```bash
javac step01_syntax_and_types/Main.java
java step01_syntax_and_types.Main
```
